markxiv

Markdown-formatted arXiv papers.

How it works

Replace arxiv with markxiv in any arXiv URL to get the paper as clean Markdown.

arxiv.org/abs/1706.03762 ↓ markxiv.org/abs/1706.03762

Free and open source.

MCP server

Use markxiv as an MCP tool in any MCP client so agents can convert arXiv papers to Markdown directly.

Install

# requires pandoc and poppler-utils (pdftotext)
cargo install --git https://github.com/tonydavis629/markxiv --path mcp

Configure

Add to your MCP config:

{
  "mcpServers": {
    "markxiv": {
      "command": "markxiv-mcp"
    }
  }
}

Tools

  • convert_paper — convert an arXiv paper to Markdown (e.g. paper_id: "1706.03762")
  • get_paper_metadata — get title, authors, and abstract
  • search_papers — search arXiv by keyword

See the MCP README for full documentation.
