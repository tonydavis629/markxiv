# markxiv.org

Markdown formatted arxiv papers

## How it works
Simply replace the 'arxiv' in an arXiv URL with 'markxiv' to get the Markdown version

### `arxiv.org/abs/1234.56789` `==>` `markxiv.org/abs/1234.56789`
